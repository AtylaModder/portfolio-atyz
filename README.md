# Portfólio Atyla Smith

Portfólio pessoal em Astro para apresentar o trabalho de Atyla Smith (Atyz) em Minecraft Bedrock, com foco em texturas, PBR, iluminação, Vibrant Visuals e direção visual.

## Stack

- Astro
- Tailwind CSS via plugin do Vite
- Swiper para o carrossel de modelos
- TypeScript

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Onde editar

- Conteúdo principal: src/data/site.ts
- Página inicial: src/pages/index.astro
- Componentes: src/components
- Tema global: src/styles/global.css

## Placeholders que precisam ser substituídos

- Links de contato
- Depoimentos reais
- Números da seção About
- Domínio em astro.config.mjs
- Imagens pessoais, renders, logos e screenshots

## Observações

- O site foi criado sem inventar clientes, números, anos de experiência ou depoimentos.
- Onde faltava informação, foram usados placeholders visíveis.
- Antes de publicar, substitua o domínio placeholder e conecte o formulário a um serviço real.
